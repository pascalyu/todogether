# todogether

Web app fait en **trois jours** : todolist pour un groupe de personnes. me contacter pour accéder à ce site.

# fonctionnalités
* inscription/connexion
* créer un groupe
* créer une tâche pour le groupe
* supprimer une tâche(softdelete)
* compléter une tâche
* undo une tâche
